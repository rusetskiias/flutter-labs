#To-do list - Список дел с погодой и курсом валют


## Краткое описание
To-do list — это простое мобильное приложение для управления делами. Пользователь может добавлять, отмечать выполненные и удалять задачи. Для дополнительной информации на втором экране отображаются погода и курс валют, загружаемые из открытого API.


## Основные функции
1. Добавление задачи - Пользователь вводит текст и нажимает кнопку «Добавить».
2. Просмотр списка задач - Все задачи отображаются в виде списка.
3. Отметка о выполнении - Нажатие на чекбокс отмечает задачу как выполненную.
4. Удаление задачи - Свайп или кнопка «Удалить» убирает задачу из списка. 
5. Просмотр дополнительной информации - На втором экране отображается погода и курс валют из API. 
6. Сохранение данных - Список задач сохраняется после закрытия приложения.

##Сценарии использования
1. Добавление задачи - Пользователь вводит текст → нажимает «+» → задача появляется в списке

2. Выполнение задачи - Пользователь нажимает на чекбокс → задача зачеркивается или меняет цвет

3. Удаление задачи - Пользователь делает свайп влево по задаче → появляется кнопка «Удалить» → задача исчезает
