## Краткое описание
To-do list - Список дел с погодой. 

Это простое мобильное приложение для управления делами. Пользователь может добавлять, отмечать выполненные и удалять задачи. Для дополнительной информации на втором экране отображается погода, загружаемая из открытого API.


## Основные функции
1. Добавление задачи - Пользователь вводит текст и нажимает кнопку «Добавить».
2. Просмотр списка задач - Все задачи отображаются в виде списка.
3. Отметка о выполнении - Нажатие на чекбокс отмечает задачу как выполненную.
4. Удаление задачи - Свайп или кнопка «Удалить» убирает задачу из списка. 
5. Просмотр дополнительной информации - На втором экране отображается погода и курс валют из API. 
6. Сохранение данных - Список задач сохраняется после закрытия приложения.

## Сценарии использования
1. Добавление задачи - Пользователь видит строку на которой написано "Введите задачу", после ввода текста пользователь нажимает на кнопку "Добавить" и задача появляется в списке.

2. Выполнение задачи - Пользователь нажимает на чекбокс → задача зачеркивается и тускнеет.

3. Удаление задачи - Пользователь нажимает на кнопка мусорного ведра → задача исчезает.

4. Просмотр погоды - Пользователь нажимает на иконку солнца и у него открывается второй экран с информацией о погоде на данный момент.
